RoadGuard-AI — Real-Time Hazard Detection
A dual-input inference system fusing YOLOv8 visual detection on 1,000+ road images with real-time accelerometer sensor data to classify both visual and physical hazards simultaneously. GPS-tagged, timestamped driver alerts fire within milliseconds. Backed by a FastAPI backend, React admin dashboard, and a production Android app.
Achieved ~90% accuracy. Reduced false-positive rate measurably vs. single-modality baseline. Published to EAIS 2026 (Submission #30).
